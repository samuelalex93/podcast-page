## Sobre

Podcast Page - Aplicação front end desenvolvida em next/react na semana NLW5.

## Instalar dependências

Instalar as dependencias em cada serviço.

```sh
yarn install
```

## Executar ambiente

```sh
yarn dev
```

## Executar jsonServer

```sh
yarn server
```
